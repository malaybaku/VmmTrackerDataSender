プライバシーポリシー

本ポリシーは、VMM Tracker Data Sender（以下「本サービス」）におけるデータの取り扱いについて説明します。本サービスは、モバイル端末のカメラで顔トラッキングを行い、同一ネットワーク内のPCにトラッキングデータを送信する機能を提供します。

1. 取得・処理するデータ

  • カメラ映像: 顔トラッキングのためにモバイル端末のカメラにアクセスします。映像はデバイス内でのみ処理され、外部サーバーへの送信は行いません。
  • トラッキングデータ: 顔の表情パラメータ（BlendShape）および頭部の姿勢データを、WebRTCによりPC端末へ直接送信します。この通信はサーバーを経由しません。
  • シグナリング情報: PC端末との接続を確立するため、暗号化された接続情報をFirebaseサーバーを経由して中継します。

2. データの暗号化と保持

シグナリング情報はクライアントサイドでAES-128-GCM暗号化を行ったうえでサーバーに送信されます。暗号鍵はPC・モバイル間でのみ共有され、サーバー管理者を含む第三者が内容を復号することはできません。

シグナリング情報は接続確立のためにのみ使用され、永続的な保存は行いません。

3. 利用する第三者サービス

  • GitHub Pages: 本サービスのWebページのホスティングに使用します。アクセス時にIPアドレス等の情報がGitHubにより記録される場合があります。詳細はGitHubのプライバシーポリシーをご確認ください。
  • Firebase (Google Cloud): シグナリング情報の一時的な中継に使用します。詳細はFirebaseのプライバシーポリシーをご確認ください。

4. お問い合わせ

本ポリシーに関するお問い合わせは、GitHubリポジトリのIssueにてお願いいたします。
